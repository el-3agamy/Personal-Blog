# Portfolio Website

A personal portfolio website built with HTML5 semantic tags and CSS3. This website showcases the skills, projects, and experiences of the developer in a clean, responsive design using modern web technologies.

## Features

- **About Me Section**: Introduction to the developer with a brief bio.
- **Skills Section**: A display of technical skills with icons or bars for better visualization.
- **Projects Section**: A collection of personal or professional projects, each with descriptions and links.
- **Contact Section**: A form where users can send messages or connect via social media.
- **Responsive Design**: The website adapts to different screen sizes (mobile, tablet, desktop) using CSS media queries.

## Technologies Used

- **HTML5** - For semantic markup (using tags like `<header>`, `<nav>`, `<section>`, `<footer>`).
- **CSS3** - For styling and creating a responsive layout.
- **Flexbox** - Used for layout structure and alignment.
- **Media Queries** - Ensures the portfolio is mobile-friendly.
